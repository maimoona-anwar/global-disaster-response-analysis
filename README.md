# Global Disaster Response Analysis (2018-2024)

## Overview
This project analyzes global disaster data from 2018 to 2024 to understand patterns in disaster impact, casualties, economic losses, and response factors.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Work Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization
- Feature analysis
- Random Forest Regression model

## Key Insights
Analyzed relationships between disaster severity, casualties, response time, aid distribution, and economic impact.
